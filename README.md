# Currency Converter

A simple app created in Flutter to convert from one currency to another with ease. 


## Dependencies 

Dependencies used are: 
 - http
 - flutter_cache_manager
 - flutter_money_formatter
 - url_launcher
 - linkable
 - flutter_launcher_icons
 - cupertino_icons
